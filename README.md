### Run Locally

To run **Dashboard chart** locally, run this command on your git bash:

Linux and macOS:

```bash
sudo git clone https://github.com/codewithsadee/dashboard-chart.git
```

Windows:

```bash
git clone https://github.com/codewithsadee/dashboard-chart.git
```
